## GCP Commands
**run tests**
```
npm test
```

**run node app**
```
node start

```
**npm packages**
```
npm help json
npm install <pkg> --save

npm install --save @google-cloud/storage
npm install --save mocha chai
```

**GCP**
```
set GOOGLE_APPLICATION_CREDENTIALS=[PATH]
gcloud app deploy
gcloud app browse
```
